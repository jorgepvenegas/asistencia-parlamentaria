## Getting Started

### What is QuienAtiende?

QuienAtiende is a citizen transparency platform that tracks the attendance of elected officials in parliamentary sessions. You can:

- **View yearly attendance** for all politicians
- **Compare performance** between representatives
- **Track monthly details** to see when absences occurred
- **Search by name** to find specific politicians
- **Filter by party** to see party-level trends
- **Access on mobile** from any device

### Where Do I Start?

1. Visit the QuienAtiende website (home page)
2. See the yearly attendance overview with all politicians
3. Click on any politician's name to see detailed history
4. Use search and filters to narrow down results

## Finding Politicians

### By Search

**How to search**:
1. Click the search box at the top of any page ("Buscar diputado...")
2. Type the politician's name (first name, last name, or part of name)
3. Matching results appear below as you type
4. Click on a name to view their profile

**Search Tips**:
- Works with partial names: "Mar" finds "María García"
- Not case-sensitive: "JUAN" = "juan" = "Juan"
- Shows up to 10 results (most relevant first)
- Requires at least 2 characters

### By Party Filter

**How to filter by party**:
1. Look for "Filtrar por partido" section on the home page
2. Check the box next to your party of interest
3. Politicians list updates instantly
4. Check multiple parties to compare
5. Click "Limpiar filtros" (Clear filters) to reset

## Viewing Attendance

### Yearly Overview

**On the home page, you'll see**:
- All politicians ranked by attendance percentage
- Color-coded attendance breakdown (attended/absent/excused)
- Summary statistics (total sessions, top/bottom performers)
- Charts showing attendance distribution

**Key Metrics**:
- **Attendance %**: Percentage of sessions attended
- **Present**: Sessions where politician was present
- **Absent**: Sessions missed without excuse
- **Excused**: Sessions missed with valid reason

### Monthly Breakdown

**To see monthly details**:
1. Click on a politician's name to open their profile
2. Scroll to "Desglose mensual" (Monthly breakdown)
3. See attendance % for each month
4. Click on a month to see daily records

**What you'll see**:
- Month name and year
- Attendance percentage for that month
- Counts of present/absent/excused

### Daily Records

**To see specific dates**:
1. Open a politician's profile
2. Scroll to "Historial reciente" (Recent history)
3. See up to 30 most recent records
4. Each record shows date and status

## Filtering & Sorting

### Sort by Attendance

**On the yearly view**:
- Politicians default to **highest to lowest** attendance
- Click column headers to re-sort (if available)
- Use the sorting dropdown for other options:
  - Highest attendance first
  - Lowest attendance first
  - Alphabetical by name
  - Reverse alphabetical

### Filter by Multiple Parties

1. Check multiple party boxes in "Filtrar por partido"
2. List automatically updates to show only selected parties
3. Useful for comparing political movements
4. Clear all with "Limpiar filtros" button

### Combine Filters

- Filter by party **AND** sort by attendance
- Example: "Show PAC politicians, sorted by attendance"

## Understanding Attendance Status

### The Three Statuses

| Status | Icon | Meaning | Color |
|--------|------|---------|-------|
| **Presente** (Present) | ✓ | Attended the session | Green |
| **Ausente** (Absent) | ✗ | Missed without excuse | Red |
| **Justificado** (Excused) | ⊘ | Missed with valid reason | Amber/Orange |

### What Counts as a Session?

A "session" is a parliamentary gathering where attendance is recorded. This typically includes:
- Regular legislative sessions
- Committee meetings
- Special sessions

### What Reasons Are Recorded?

Common excused absence reasons include:
- Medical appointments/illness
- Death in family
- Official duties elsewhere
- International travel
- Official leave

## FAQ

### General Questions

**Q: How often is data updated?**
A: Attendance data is updated daily at 3 AM (Pacific Time) with the latest parliamentary records.

**Q: Can I download the data?**
A: Currently, data can only be viewed through the website. Data export may be available in future versions.

**Q: Who maintains this data?**
A: QuienAtiende team works with official parliamentary records to ensure accuracy.

**Q: Is this data official?**
A: Yes, all attendance data comes from official parliamentary sources and records.

### Finding Information

**Q: I can't find a politician. Why?**
A:
- Check spelling of name
- Politicians are only listed if active during that year
- Try searching by last name first
- Some officials may have left office

**Q: How do I search for politicians by party?**
A: Use the "Filtrar por partido" filter on the home page, or visit a politician's profile to see their party.

**Q: Can I see data for previous years?**
A: Currently, 2026 data is displayed. Previous years will be available as the platform expands.

### Understanding Data

**Q: What's the difference between "Absent" and "Excused"?**
A:
- **Absent**: Missed the session with no recorded reason
- **Excused**: Missed the session but provided a valid reason (medical, family emergency, etc.)

**Q: How is the attendance percentage calculated?**
A: Attendance % = (Sessions Attended) / (Total Sessions) × 100%

Example: 171 attended ÷ 200 total = 85.5%

**Q: Why does attendance vary so much by month?**
A:
- Parliament takes breaks during holidays and recesses
- Fewer sessions = smaller monthly totals
- Some months have more committee work than plenary sessions

### Troubleshooting

**Q: The page loads slowly on my phone.**
A:
- Check your internet connection
- Try WiFi instead of mobile data
- Refresh the page
- Clear browser cache (Settings → Clear Data)

**Q: Charts aren't showing. Why?**
A:
- JavaScript may be disabled (enable in settings)
- Try a different browser
- Clear cache and refresh

**Q: Search results aren't appearing.**
A:
- Type at least 2 characters
- Check for spelling
- Try searching by last name

**Q: I found incorrect data. What do I do?**
A: Report issues at [contact email or GitHub link]

## Accessibility

### For All Users

QuienAtiende is designed to be usable by everyone:

**Color Contrast**:
- All text meets WCAG AA standards (4.5:1 ratio)
- Readable in low light conditions
- Not reliant on color alone to convey information

**Touch Friendly**:
- All buttons are at least 44×44 pixels (easy to tap)
- Links are properly spaced
- No hover-only features on mobile

**Readable Text**:
- Minimum 12px font size
- Clear fonts (sans-serif)
- Sufficient line spacing

### Keyboard Navigation

You can navigate QuienAtiende entirely with keyboard:
- **Tab**: Move to next element
- **Shift+Tab**: Move to previous element
- **Enter**: Activate button or link
- **Arrow Keys**: Navigate menus and lists
- **Esc**: Close dropdowns/modals

### Screen Reader Support

QuienAtiende works with screen readers:
- Semantic HTML for proper structure
- Form labels associated with inputs
- Image alternatives for icons
- Table headers properly marked

**Tested with**:
- NVDA (Windows)
- JAWS (Windows)
- VoiceOver (macOS/iOS)
- TalkBack (Android)

### Adjusting Text Size

Most browsers let you zoom:
- **Windows/Linux**: Ctrl + (zoom in) | Ctrl - (zoom out)

- **Mobile**: Pinch to zoom

If your device has dark mode enabled, QuienAtiende will display in dark colors automatically.

To override:
- Check browser settings
- Some devices allow app-level dark mode toggle
